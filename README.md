# メディア閲覧しつつ整理はかどりソフト

## 概要
- 画像ファイルを閲覧しながら、ショートカットキーに割り当てたフォルダにどんどん整理するためのソフトです。

### 上側のツリービュー
- サイドビューの上側のツリービューが移動先（整理先）のフォルダです。ツリービューのルートアイテムをクリックすると、フォルダ選択ダイアログがでます。
- ツリービュー上で右クリックすると、振り分け先フォルダに対応させたいショートカットキー（F1～F5)を選べます。
- <img width="345" height="314" alt="image" src="https://github.com/user-attachments/assets/161c3ccb-32c9-4ff0-a478-b5e39a834d13" />

### 下側のツリービュー
- サイドビューの下側のツリービューが移動元のフォルダです。
- ツリービュー上にはフォルダまでしか表示されません。
- 上下キーでツリービュー上のアイテム選択を移動します。

### 閲覧＆整理の流れ
- 画面右の表示領域に閲覧したいファイルをドロップすると表示されます。
- あるいは、下側ツリービューのフォルダを選択すると、そのフォルダ内の最初の画像かZip画像が表示されます。
- 上側ツリービューで、移動させたいフォルダを選択しておくか、ショートカットキーを設定しておきます。
- 画像表示中にスペースキーを押すと、整理先ツリービューで選択中のフォルダに閲覧中のファイルが移動します。
- あるいは、登録したショートカットキー、例えばF1を押すとF1に割り当てている遷移先フォルダに、閲覧中のファイルが移動します。
- 画像表示領域は、今まで閲覧していたファイルと同じフォルダ配下の次のファイルが自動でロードされます。
- 同じフォルダ内の次の画像を見たい場合はPageDown,前の画像を見たい場合はPageUpです。
- 書庫内の画像を送りたい場合は左キー、戻りたい場合は右キーです。

### キー操作まとめ
- スペースキー：現在表示中のファイルを、遷移先ツリーの選択中フォルダに移動させる。
- pageup/pagedown：現在閲覧中のファイルと同じフォルダ内の次/前のファイルを開く。
- 左/右：現在閲覧中の書庫の次ページ/前ページを開く。
- F1～F5：整理先フォルダで割り当てている場合、現在閲覧中のファイルを割当先のフォルダに移動させる。
- ESC：現在閲覧中のファイルをアンロードする。

## 設計を頑張ったポイント
- 表示できるメディアをいろんな型に対応できるようにクラス設計をできるだけ抽象化しました。
- 現在は通常画像とZip画像のみ対応ですが、gifやmp4,wmvなど、対応させたいメディアの種類が増えた場合でも、根幹のソースは変更なしで対応可能なように設計してあります。
